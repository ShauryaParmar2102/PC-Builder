# PC Builder Web Application

### Description

A full-stack PC Builder web application that allows users to configure custom PC builds using dynamic components, calculate real-time build costs, and save builds locally or to a backend showcase system. This project demonstrates CRUD operations, REST API integration, and structured cost calculation logic across frontend and backend layers.

# 🛠️ Tech Stack

🎨 Frontend: HTML, CSS, JavaScript (Vanilla JS)

🖥️ Backend: Node.js + Express.js

📦 Data: JSON-based storage (or localStorage for drafts)

🧱 Architecture: MVC (Model–View–Controller)

🚀 How to Run the Project

1️⃣ Download the Project

1. Click the green Code button on GitHub

2. Select Download ZIP

3. Extract the ZIP file to your computer

4. Open the project folder in Visual Studio Code

 2️⃣ Install Dependencies

 1. Open a terminal in the project folder (where package.json is located):

 2. npm install

 3. This will install all required packages and create the node_modules folder.

3️⃣ Start the Server

npm run start

4️⃣ Open the App

## Go to your browser:

http://localhost:8000
